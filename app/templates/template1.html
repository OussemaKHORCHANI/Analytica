State Profiling By Hygiene

We used PCA (Principal Component Analysis) algorithm to analyze the correlation between the different states (individuals), the different hygiene variables and between individuals, variables. Here are some observations from our graph :

• California State is positively correlated with the variables Obese Population, Total Number of Restaurants, Food Insecurity Population and Low Access Population.

• North Carolina State, Ohio State, NewYork State are positively correlated with the Total Number of Gym.

• We can say that the variables Total Number of Gym, Diabete Population, Resto sales Per Capita, Direct sales Farms and FreshVegetables Farms are positively correlated with each other.

State Profiling By Food Access

We used PCA (Principal Component Analysis) algorithm to analyze the correlation between the different states (individuals), the different food access variables and between individuals, variables. Here are some observations from our graph :

• Georgia State and Missouri State are positively correlated with the variables Poverty Rate, Benefit Assistance Program per Capita and Redemptions Assistance Program per Store.

• the variables Poverty Rate, Benefit Assistance Program per Capita and Redemptions Assistance Program per Store are orthogonally correlated with all the other variables which are positively correlated with each other.

State Segmentation

We used the HAC (hierarchical cluster analysis) algorithm to segment the states into 4 major groups while keeping the states closest to them in the same group. Using the ward.D2 distance as a metric we visualized the dendrogram cluster to get an idea of ​​the states that have the same behavior.
